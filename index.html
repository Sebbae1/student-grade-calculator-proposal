Class Product Proposal - Student Grade Calculator

Application Overview: A webpage designed to help student's calculator their letter grade through their percentages.
Technology Overview: The usage of HTML, CSS, and JavaScript. Github will be used to host the webpage.
Proposal Last Updated: Thursday, November 2nd, 2023
Feature List: Options and selections will be used to display information. HTML and CSS wil be used to design and display the webpage accordingly.
Future Feature List: The possibility of transferring the user to a different webpage whenever a grading scale is selected.
Basical Technical Features: The use of button(s) in the webpage. The options and select ids will be used. HTML webpage layout will be used to move certain text to certain areas throughout the webpage.
Advanced Technical Features: The usage of possible tables that contain different grading scales. The usage of different possible calculations used to determine the letter grade based off of a student's percentage.
